# Requirements

- Java_Platform_ME_SDK (My Version: 3.0)
- NetBeans (My Version: 8.2)
- Java ME plugin in Netbeans (My Version: 1.35.1)
- Sun Java Wireless Toolkit (My Version: 2.5.2)
